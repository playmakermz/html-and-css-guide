## Cara instalasi react pada Ubuntu dan WSL 

```
sudo apt update
sudo apt install nodejs npm
```

Install react globally pada node

```
sudo npm install -g create-react-app
```

buat folder khusus untuk app react
```
mkdir react-test
cd react-test
```

buat project react 
```
npx create-react-app my-app
cd my-app
npm start
```
